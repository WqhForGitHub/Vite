📦 ESM vs CommonJS Demo

🌿 ESM（ES Modules）

// math.mjs
export const add = (a,b) => a+b
export default { add }

// main.js
import { add } from './math.mjs'

静态、异步、浏览器原生支持。Vite 的核心基石。

🌳 CommonJS

// math.cjs
exports.add = (a,b) => a+b
module.exports = { add }

// main.js
const { add } = require('./math.cjs')

动态、同步加载，Node.js 早期默认方案。
