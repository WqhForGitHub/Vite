使用 CSS 变量实现的多主题切换

通过 data-theme 属性 + CSS 自定义属性的方式切换主题， 切换瞬时无闪烁，主题选择会持久化到 localStorage。

🎨 颜色变量

所有颜色均通过 CSS 变量定义，切换主题只需更换变量集。

  • --c-primary
  • --c-bg
  • --c-surface
  • --c-text
  • --c-border
