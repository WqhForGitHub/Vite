⚔️ Rollup vs Vite 对比

📦 Rollup

定位：纯打包工具（Bundler）

  • 专注于 library 输出，ESM 友好
  • 插件生态成熟（@rollup/plugin-*）
  • Tree-shaking 业界标杆
  • 不提供 dev server / HMR（需配合插件）

⚡ Vite

定位：构建工具链（Dev Server + Bundler）

  • Dev：基于浏览器原生 ESM + esbuild 预构建
  • Build：底层使用 Rollup
  • 开箱即用：HMR / TS / JSX / CSS Modules / 静态资源
  • 面向应用开发，但也能打包 lib（build.lib）
